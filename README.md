# Página académica de Francis Mora Ferreras

Sitio estático preparado para GitHub Pages.

## Archivos

- `index.html`: contenido principal de la página.
- `styles.css`: diseño visual y versión móvil.
- `script.js`: menú móvil, año automático y navegación activa.
- `assets/research-visual.png`: imagen visual de fondo para la portada.

## Publicación en GitHub Pages

1. Crear un repositorio público llamado `FMoraFerreras.github.io`.
2. Subir estos archivos a la raíz del repositorio.
3. Entrar en `Settings > Pages`.
4. Seleccionar `Deploy from a branch`.
5. Usar la rama `main` y la carpeta `/root`.
6. La página quedará disponible en `https://FMoraFerreras.github.io`.
